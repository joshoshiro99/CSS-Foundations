-From inside to outside, what is the order of box-model properties?
content, padding, border, and margin.
-What does the box-sizing CSS property do?
box-sizing allows the developer to set the box sizing calculation model between standard and
alternative.
-What is the difference between the standard and alternative box model?
Standard allows the content, padding and border add to the total size of the display box. (width property
is calculated from the other properties)
Alternative box model has the content padding and border subtract from the total display size. (width 
property sets the max)
-Would you use margin or padding to create more space between 2 elements?
margin creates space between elements
-Would you use margin or padding to create more space between the contents of an element and its border?
padding creates space between the border and content of an element
-Would you use margin or padding if you wanted two elements to overlap each other?
negative margin allows for two elements to overlap each other
