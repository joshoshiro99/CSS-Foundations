-How do you select a specific element on your page with your browser’s developer tools?
Right click -> Inspect on the specific element to select that element.
-What does a strikethrough in a CSS element mean in your browser’s developer tools?
It means that css element is overwritten by another element in the hierarchy.
-How do you change CSS in real time on specific elements of a web page with your browser’s developer tools?
After right click -> Inspect on the element to edit; you can change the css in the styles tab of the elements panel.
